# BillSavers

A simple and efficient web application for tracking bills and managing savings goals.

## Features

- **Bill Tracking**: Keep track of all your bills and due dates
- **Savings Calculator**: Calculate and visualize your savings goals
- **Dashboard**: View all your financial information at a glance
- **Settings**: Customize your experience and manage preferences

## Pages

- **index.html** - Homepage and main dashboard
- **bills.html** - Bills tracking page
- **savings.html** - Savings calculator page
- **settings.html** - User settings and preferences
- **css/style.css** - Styling for all pages
- **js/script.js** - JavaScript functionality

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/srichandkotha/BillSavers.git
   ```

2. Open `index.html` in your web browser

## Usage

- Navigate through the application using the top menu
- Add, edit, and delete bills
- Track your savings goals
- Manage your settings

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage for data persistence
